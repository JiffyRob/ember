==================
:mod:`ember.ui.TextField`
==================

.. currentmodule:: ember.ui

.. autoclass:: TextField

    A TextField is an interactive :ref:`element`.

    Internally, the TextField uses a :ref:`Text<ui-text>` object contained
    within a :ref:`VScroll / HScroll<ui-scroll>` object.
